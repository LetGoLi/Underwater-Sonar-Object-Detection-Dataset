# Underwater-Sonar-Object-Detection-Dataset
* This dataset is introduced in our submitted manuscript: FA-YOLO: A Lightweight Frequency-Aware Framework for Underwater Object Detection on the USOD Dataset. By Zhicheng Li, Bin Chu, Zhipan Wang, Nan Wei, Weijie Zhou
* This dataset is the largest underwater sonar detection dataset, containing 17,533 sonar images and 5 categories. It consists of forward-looking sonar images and side-scan sonar images, and supports annotations in both COCO and YOLO formats.
